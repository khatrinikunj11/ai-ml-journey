# Day 1

Today I learnt about modules,variables,datatypes,strings

I will make 5 projects today with explanation listed below-

## Project 1 - Temperature Converter with Formatting

Write a program that asks the user for a temperature in Celsius and converts it to both Fahrenheit and Kelvin. Display the results using string formatting with escape sequences to make the output neat (e.g., tabs, new lines).

## Project 2 - Name Manipulator

Create a program that takes a full name as input (e.g., "john doe smith") and outputs it in the following formats:

- Title case (First letter of each word capitalized)
- All uppercase
- All lowercase
- Count of total characters (excluding spaces)
- Initials (e.g., "J.D.S.")

## Project 3 - Email Validator

Write a program that takes an email address as input and checks:

- If it contains "@" symbol
- If it ends with ".com" or ".in"
- Extract and display the username (part before @)
- Extract and display the domain name (part after @)

## Project 4 - String Reverser

Create a program that takes a sentence as input and:

- Reverses the entire string
- Reverses each word individually but keeps them in the same order
- Prints every alternate character 

## Project 5 - Password Strength Checker

Write a program that takes a password as input and checks:

- Length is at least 8 characters
- Contains at least one digit
- Contains at least one uppercase letter
- Display appropriate messages for each condition
